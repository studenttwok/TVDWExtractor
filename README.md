# TVDWExtractor
TVDW Game Data Extractor

This cam be used to extract the game data from the game 'TV Dream Works'
You need to have python 2.7 installed in the computer in order to use this program.

It actually extracts:
pdata/film.pgm
pdata/person.dat
pdata/pintro.dat
pdata/script.pgm
table/fresh.tab
